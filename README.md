# Documents Repository

## Purpose
This is the primary repository for all official **System Design** artifacts. It contains the core deliverables required for the SENG 31242 course.

## Directory Structure
- `pitch/`: Idea Pitch Deck and Project Approval Form.
- `srs/`: Software Requirements Specification (Markdown source + PDF).
- `sds/`: Software Design Specification (Markdown source + PDF).
- `planning/`: Project Timeline and Gantt charts.
- `architecture/decisions/`: Architectural Decision Records (ADRs).
- `meetings/`: Meeting notes and logs (committed within 24 hours of meeting).
- `presentation/`: Final Design Presentation materials.

## Tooling
- Markdown for document source.
- `draw.io` / `PlantUML` for diagrams (source + exports required).
- PDF generators.

## Standards
- **Diagrams**: Both source files (e.g., `.drawio`) and exported images (SVG/PNG) must be committed.
- **Meeting Notes**: Must be committed to the `meetings/` folder within 24 hours.
- **ADRs**: Use the template provided in the course guidelines.
